1. Func: print()
syntax: 

```py
    print(<object>, sep = <seperator>, end = <end>)
    # Object: the object wanna print, mutiple objects 
    # sep: the seperator - kí tự phân các giữa các object muốn in
    # end: default end = "/n" - end line - mặc định cuối object muốn in là kí tự xuống dòng  
```
